Macro Cheat Sheet

Rules

  • Only tokens are executed. Free text is ignored.
  • Tokens are case-insensitive.
  • Tokens are separated by whitespace; parentheses and quotes keep arguments together. For example:
DELAY(100)
TAP(DOWN)
UP(SHIFT)
REPEAT(5, TAP(SPACE))

is equivalent to: DELAY(100) TAP(DOWN) UP(SHIFT) REPEAT(5, TAP(SPACE))

Delays are in milliseconds.

Commands

  • DELAY(ms) / SLEEP(ms) — pause for ms.
  • REPEAT(n, cmd1, ...) — repeat the sequence n times.
  • TEXT(...) — type the given text exactly. Works with quotes and across lines:
TEXT(hello)
TEXT(
  "hello"
)
  • TAP(KEY) — press & release a key. TAP(KEY*x) — tap key x times.
  • HOLD(KEY) / RELEASE(KEY) — hold/release a key.
  • RANDOM(min,max) — prints a random integer. Any numeric argument may be a range: DELAY(RANDOM(100, 250)), REPEAT(RANDOM(2, 5), TAP(ENTER)).

Keys

Letters A..Z, digits 0..9, arrows LEFT/RIGHT/UP/DOWN, ENTER, TAB, ESC, SPACE, BACKSPACE, DELETE, HOME, END, PAGEUP, PAGEDOWN, PRINTSCREEN, SCROLLLOCK, PAUSE, INSERT, function keys F1..F12, keypad KP/, KP*, KP-, KP+, KPENTER, KP..

To chord modifiers, combine holds with taps, e.g.: HOLD(CTRL) TAP(S) RELEASE(CTRL).

Examples

DELAY(300) TEXT("hello") TAP(ENTER)
REPEAT(3, DELAY(RANDOM(50, 100)), TEXT("hello"), TAP(SPACE), TEXT("world"), TAP(ENTER))
HOLD(SHIFT) TAP(A) RELEASE(SHIFT)
TAP(DOWN*5) DELAY(150) TAP(ENTER)

Tip: split your script across lines for readability; whitespace does not matter.
